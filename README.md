# necotube 🐾🎬

necotube は **YouTube の非公式Webクライアント**です。
Youtubeでできることはだいたい全てできます。
主に規制回避です。インスタンスの作り方は以下

---

* **Next.js**
* **YouTube Data API**
を使用してます。

---

## 🔑 環境変数

YouTube Data API を利用する場合、以下の環境変数が必要です。

```
YOUTUBE_API_KEY=xxxxxxxxxxxx
```

---

## ⚠️ 注意事項

* YouTube の仕様変更により、予告なく動作しなくなる可能性があります
